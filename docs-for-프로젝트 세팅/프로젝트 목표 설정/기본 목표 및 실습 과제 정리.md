# 프로젝트 기본 목표 및 실습 과제 정리

## 프로젝트 목적

Twilio Voice API를 활용한 미니 콜센터 시스템을 직접 구현해보며,
CTI(Computer Telephony Integration) / ACD / IVR 도메인을 실습 수준에서 익히는 것.

채용 시장에서 요구하는 콜센터 플랫폼 개발 역량(인바운드/아웃바운드, 상담 이력, 실시간 모니터링)을
소규모 프로젝트로 체험하는 것이 핵심 목표.

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| 백엔드 | Spring Boot 4, Java 21, JPA, PostgreSQL |
| 프론트 | Next.js 16, React 19, TanStack Store, Tailwind v4 |
| 통신 | Twilio Voice SDK, TwiML, Twilio REST API |
| 실시간 | Spring WebSocket + STOMP |
| 인증 | JWT (구현 완료) |

## 핵심 학습 포인트

| 키워드 | 내용 |
|--------|------|
| TwiML | Twilio Markup Language — 통화 흐름 제어 XML |
| Webhook | Twilio가 이벤트 발생 시 서버로 HTTP 요청 전송 |
| CTI | 전화 시스템과 컴퓨터 시스템 연동 |
| ACD | 자동 통화 분배 — 상담사 대기열 배분 |
| IVR | 자동 응답 시스템 — 번호 입력으로 메뉴 분기 |
| StatusCallback | 통화 상태 변화(ringing → in-progress → completed)를 서버가 수신 |
